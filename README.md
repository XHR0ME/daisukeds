Features 7 different vocal modes:

- Standard - Only chest range, legacy data
- Classic - Calm tone with falsetto high notes
- Warm - More soft and cute voice, legacy data
- Vivid - A youthful strong sounding voice with big high range, legacy data
- Powerful - More strong and masculine sound, legacy data
- Falsetto - High ranged airy data, legacy data
- Mature - Deep and masculine strong voice
