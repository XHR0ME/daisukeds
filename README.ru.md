Мужской виртуальный певец с ярким голосом. Разнообразные вокальные режимы позволяют добиться разных эмоций на семи различных языках!

Родной язык: русский

Набор данных включает: русский, японский, английский

Имеет 7 различных вокальных режимов:
- Standard — только грудной диапазон, устаревшие данные
- Classic — спокойный тембр с фальцетными высокими нотами
- Warm — более мягкий и милый голос, устаревшие данные
- Vivid — молодое сильное звучание с широким диапазоном, устаревшие данные
- Powerful — более сильное и мужественное звучание, устаревшие данные
- Falsetto — воздушные высокие ноты, устаревшие данные
- Mature — глубокий мужественный сильный голос
